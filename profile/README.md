## 📖 Introduction
Sierra Chart for Windows is a desktop charting and trading platform built for people who spend their day reading markets. It renders both historical and live data with very little lag, so charts stay smooth even with many symbols and studies loaded. What sets it apart is its depth: footprint order flow charts, a large collection of studies, and the ability to write custom indicators. If you want a lightweight, reliable Windows tool that puts charting first, Sierra Chart is designed for exactly that.

## ✨ Highlights

### Charting and Analysis
- **Footprint / numbers bars** — Inspect traded volume at each price to follow order flow inside every bar.
- **High-speed rendering** — Scroll and zoom across large datasets without the chart bogging down.
- **Broad indicator library** — Use a wide range of Sierra Chart indicators, from moving averages to advanced studies.

### Trading and Customization
- **Custom studies** — Extend the platform with your own studies when the built-in tools are not enough.
- **Chart order entry** — Place and manage orders straight from the chart with configurable controls.
- **Personalized layouts** — Save colors, timeframes, and workspaces so your environment loads the same way each time.

## 💻 System Specs

**Minimum**
- OS: Windows 10 64-bit
- Processor: A modern multi-core 64-bit processor
- Memory: 4 GB RAM
- Storage: A few hundred MB free, plus space for market data

**Recommended**
- OS: Windows 11 64-bit
- Processor: A recent, faster multi-core 64-bit processor
- Memory: 8 GB RAM or more
- Storage: An SSD with several GB free for historical data
